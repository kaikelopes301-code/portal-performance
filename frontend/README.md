# Sistema de Faturamento - Frontend

React + Vite frontend otimizado para leveza e performance.

## Setup

```bash
npm install
```

## Executar

```bash
npm run dev
```

Acesse: `http://localhost:5173`

## Build

```bash
npm run build
```

## Tecnologias

- React 18
- Vite
- TypeScript
- TailwindCSS
- SWR (data fetching)
- Zustand (state)
- React Hook Form
- Fetch API nativo (sem axios)
